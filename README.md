## Hidove CloudFlare Panel

* Blog：<http://blog.hidove.cn>

## 简介

* 基于React的一个简易的CloudFlare API面板
* 面板有多种语言选择，除中文外，其余皆由百度翻译提供。
* 该项目为纯前端项目，需要配合后端运行
